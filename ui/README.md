# ui/ — Пользовательский интерфейс

Графический интерфейс на PySide6 (Qt 6). Полностью отделён от бизнес-логики:
обменивается с `core/` только через публичный API и EventBus.

## Структура

| Директория | Описание |
|-----------|---------|
| [`qt/`](qt/README.md) | Qt-приложение: главное окно, экраны, компоненты |

## Архитектурные принципы

- **QThread** — транскрибация выполняется в отдельном потоке, UI не блокируется
- **EventBus → BusToQtBridge** — события pipeline доставляются в UI через Qt signals (QueuedConnection)
- **QStackedWidget** — все экраны находятся в одном стеке, переключаются в `MainWindow`

## Запуск

```bash
python gui.py
```
