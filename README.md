# USSR Diplom — Транскрибатор совещаний

Десктопное приложение для локальной транскрибации аудиозаписей совещаний.
Использует [faster-whisper](https://github.com/SYSTRAN/faster-whisper) для ASR и [pyannote.audio](https://github.com/pyannote/pyannote-audio) для диаризации (разделения спикеров).
Графический интерфейс реализован на PySide6 (Qt 6).

## Возможности

- Транскрибация WAV / MP3 / M4A / OGG / FLAC без отправки данных в облако
- Диаризация: автоматическое разделение по спикерам
- Экспорт результатов в JSON и TXT
- Встроенный редактор транскрипта с временной шкалой
- Система плагинов для подключения новых моделей
- Возобновление прерванной обработки с последнего этапа

## Быстрый старт

```bash
# 1. Клонировать репозиторий
git clone <url> && cd USSR_diplom

# 2. Установить зависимости (требуется uv)
uv sync

# 3. Запустить GUI
python gui.py

# 4. Или CLI
python main.py audio.wav
```
